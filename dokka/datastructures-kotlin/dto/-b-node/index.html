BNode

class BNode<K : Comparable<K>, V>(key: K, value: V) : Node<K, V, BNode<K, V>>

Класс BNode представляет узел дерева с двумя параметризованными типами: K (ключ) и V (значение). Он наследует от абстрактного класса Node и реализует структуру узла для двоичного дерева.

Parameters

K

Тип ключа, который должен быть сравнимым (реализует интерфейс Comparable).

V

Тип значения, которое хранится в узле дерева.

key

Ключ для этого узла.

value

Значение для этого узла.
